üõ† Troubleshooting

Problem: "403 Forbidden" errors

Solution: Your session token has expired. Get a new one using Step 1.

Problem: "Login failed" on ICICI page

Solutions:

  • Verify your ICICI Direct username and password
  • Make sure your account has API access enabled
  • Contact ICICI Direct support if needed

Problem: Can't find session token after login

Solution: Look for a JSON response containing a field like "session_token" or "token". It's usually displayed on the success page.
